# Population Growth Prediction: Doha Metropolitan

This is the code to produce the binary population prediction used in the paper: 
 **"Modeling Megaproject-Induced Population Dynamics in Emerging Global Cities: An integrative spatiotemporal framework - The Case of Doha metropolitan toward 2050."**

1. The Data folder contains the Doha metropolitan population maps used to train the model and produce the prediction.
2. Run main.py to train the ConvLSTM model.
3. Run predict.py to produce predicted population maps.

This work is based on the code from [Urban-Change-Detection-Using-ConvLSTM](https://github.com/ShoumanOmar/Urban-Change-Detection-Using-ConvLSTM) by Omar Shouman.
